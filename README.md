# 📊 Data Professional Dashboard

## 📌 Overview
This repository contains a Power BI dashboard designed to analyze the demographics, skill sets, and compensation of data professionals. The `DATA PROFESSIONAL DASHBOARD.pbix` file provides actionable insights into the data industry, helping users understand salary trends, popular technologies, and career progression across different roles (e.g., Data Analyst, Data Scientist, Data Engineer).

## 📈 Key Features
* **Salary Analysis:** High-level key performance indicators (KPIs) tracking average salaries broken down by job title, experience level, and geographic location.
* **Tech Stack Trends:** Visualizations highlighting the most popular programming languages, tools, and platforms used by data professionals.
* **Demographic Breakdown:** Insights into the education levels, age distribution, and gender of survey respondents within the data field.
* **Work-Life Metrics:** Analysis of remote vs. hybrid vs. on-site work preferences and their impact on job satisfaction.
* **Interactive Filtering:** Dynamic slicers allowing users to drill down into specific job titles, countries, or salary brackets.

## 🛠️ Tools & Technologies Used
* **Power BI Desktop:** Used for data modeling, DAX measure creation, and visual reporting.
* [cite_start]**Theming:** The dashboard utilizes Power BI's built-in "Innovate" theme [cite: 130][cite_start], built upon the base "CY25SU12" theme[cite: 113], for a sleek and modern visual design.


## 🚀 Getting Started
1. **Install Power BI:** Download Power BI Desktop from the [Microsoft Store](https://powerbi.microsoft.com/desktop/) if you don't already have it installed.
2. **Download the File:** Clone this repository or download the `DATA PROFESSIONAL DASHBOARD.pbix` file directly to your local machine.
3. **Open the Dashboard:** Double-click the file to launch it in Power BI Desktop.
4. **Interact:** Click on the charts and use the slicers to cross-filter and explore specific data segments.
